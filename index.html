Intuitive

It is strongly influenced by jquery so there are many similar methods and you'll feel at easy developing with Umbrella.

However there are also some improvements to make your code concise. Check everything in the documentation:

Documentation 
// Simple events like jquery
u("button").click(function(){
  alert("Hello world");
});

// Send form through ajax when submitted
u('form.login').ajax(function(res){
  window.href = '/user/' + res.id;
});

Tiny and Performant

2kb when minimized and gzipped means it will load in a snap on mobile platforms.

The speed is similar to jquery and there are many tests, including performance ones. Check them out:

Source Tests 
it("can select by class", function() {
  expect(u('.demo').nodes.length).to.equal(1);
});

it("can select by tag", function() {
  expect(u('body').nodes.length).to.equal(1);
});
