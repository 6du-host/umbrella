Intuitive

It is strongly influenced by jquery so there are many similar methods and you'll feel at ease developing with Umbrella.

However there are also some improvements to make your code intuitive. Check everything in the documentation:

Documentation 
// Simple events like jquery
u("button").click(function(){
  alert("Hello world");
});

// Send form through ajax when submitted
u('form.login').ajax(function(res){
  window.href = '/user/' + res.id;
});

Tiny and Clear

2kb when minimized and gzipped means it will load in a snap even on mobile. The core is reusable so new features are also tiny.

Just check those two features or the source code itself, which is highly commented and documented:

Source on Github 
// Add child after all of the current nodes
u.prototype.after = function(text) {
  return this.adjacent('afterend', text);
};
// Check if any of the nodes matches the selector
u.prototype.is = function(selector){
  return this.filter(selector).nodes.length > 0;
};
